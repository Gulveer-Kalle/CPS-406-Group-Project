# CPS 406 Co-op Management System

A desktop application built with Electron, HTML, CSS, and JavaScript for managing co-op program applications, reports, and role-based dashboards for students, industry partners, and coordinators.

---

## üèó Project Overview

This project is designed to facilitate the co-op program workflow:

- **Students:** Submit applications, track status, submit reports, view grades, and see assigned coordinators and partners.
- **Industry Partners:** Manage assigned students, submit evaluations, track progress.
- **Co-op Coordinators:** Review applications, assign students to partners, track reports, and provide feedback and grades.

The application is **role-based**, meaning each type of user sees a customized dashboard.
